# Release v0.1.0

## GPT Image Waterfall 初版发布

这是项目的第一个可用版本：

- 已完成从开源 Prompt 仓库到产品化内容站的第一轮改造
- 已具备首页分发、分类浏览、详情页承接、Prompt 复制、收藏、最近浏览与基础 SEO 能力
- 已完成 README、部署文档、截图展示与 GitHub 仓库初始化

## 本版亮点

- 产品化首页而非 README 镜像
- 更像 Pinterest / 小红书风格的信息浏览结构
- 基于构建脚本统一清洗和生成站点数据
- 适合继续往专题页、标签页、收藏体系和部署上线方向推进

## 适用场景

- AI 图片 Prompt 灵感站
- Prompt 资源聚合站
- 开源内容产品化 Demo
- AI 创意内容展示型网站
